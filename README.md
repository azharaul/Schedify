Schedify is a modern, intuitive Android application designed to help students manage their class schedules efficiently. With its clean user interface and proactive notification system, Schedify ensures you never miss a lecture again.

## ‚ú® Features

- **Dynamic Schedule Management**: Easily view, add, edit, and delete your daily class schedules.
- **Smart Reminders**: Get notified 30, 20, and 10 minutes before your class starts.
- **Real-time Countdown**: A dedicated countdown timer on the home screen showing your next upcoming activity.
- **Day-based Organization**: Intuitive tab-based navigation to switch between different days of the week.
- **Material Design 3**: A beautiful, dark-themed UI following the latest Android design principles.
- **Undo Actions**: Accidentally deleted a schedule? Quickly restore it with the built-in undo feature.

## üöÄ Getting Started

### Prerequisites

- Android Studio Flamingo or newer
- Android SDK 34+
- A physical device or emulator running Android 8.0 (Oreo) or higher

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/Schedify.git
   ```
2. Open the project in **Android Studio**.
3. Let Gradle sync and build the project.
4. Run the app on your device/emulator.

## üì± Permissions

The app requires the following permissions:
- `POST_NOTIFICATIONS`: To show reminders (Android 13+).
- `SCHEDULE_EXACT_ALARM`: To trigger reminders at precise times.
